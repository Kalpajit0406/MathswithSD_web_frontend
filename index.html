THE MONOREPO BREAKDOWN

Architected for Scalability & Stability

Our codebase is carefully partitioned into isolated sub-projects inside a single, coordinated monorepo for maximum operational readiness.

FLUTTER FRONTENDS Cross-Platform UI
  • Student Client: Native speed assessment player with zero-lag custom math layouts.
  • Educator Client: Mathpix camera scanning module and dynamic examination builder.
  • Rendering Engine: Custom canvas-painted mathematical formula renderings.
NODE.JS BACKEND API & Integrations
  • OCR Ingestion: Streamlined pipeline sending image streams to the Mathpix OCR engine.
  • Security Protocols: Encrypted JWT sessions with strict NoSQL payload filters.
  • Chapter Management: Regex-based Chapter Normalization engine to prevent duplicates.
DATABASE & SYNC Data Layer
  • MongoDB Storage: Schema-verified collections for questions, assessments, and users.
  • Optimized Queries: Custom indexing on verification queues and active exam session IDs.
  • Polled Version Caching: Dynamic version-token sync reducing server loads by 80%.
